# Newsletter-signup
Newsletter Signup

A simple and responsive Newsletter Signup web application built with Node.js, Express.js, and Mailchimp API. Users can subscribe to a mailing list through a clean signup form, and their information is automatically added to a Mailchimp audience.

🚀 Features
Responsive newsletter signup form
Collects first name, last name, and email address
Integrates with the Mailchimp Marketing API
Success and failure pages for better user experience
Built with Express.js
Easy to deploy on platforms like Render

⚙️ Installation
Clone the repository:
git clone https://github.com/your-username/Newsletter-Signup.git
Navigate to the project directory:
cd Newsletter-Signup
Install dependencies:
npm install
Configure your Mailchimp API credentials in your application.
Start the server:
node app.js

Or, if you're using Nodemon:

nodemon app.js
Open your browser and visit:
http://localhost:3000
